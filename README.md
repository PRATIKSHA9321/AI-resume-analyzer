# AI Resume Analyzer

## 📌 Project Description

An AI-based resume analyzer that evaluates resumes, extracts key skills, and provides suggestions to improve job readiness. The system helps users identify missing skills and enhance their profiles.

---

## 🚀 Features

* Upload resume (PDF format)
* Extract skills automatically
* Analyze resume content
* Suggest improvements
* Store and view previous results

---

## 🛠️ Technologies Used

* Python
* Flask
* HTML, CSS
* SQLite Database

## ▶️ How to Run

1. Clone the repository
2. Install required libraries
3. Run the app:

```
python app.py
```

4. Open browser:

```
http://127.0.0.1:5000
```
